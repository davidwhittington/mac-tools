macOS Utilities

Standalone tools.
Install what you need.

A collection of independent macOS utilities for privacy, traffic generation, system maintenance, and development setup. Each tool installs separately via Homebrew.

Add the tap
brew tap davidwhittington/mac-tools
Then install individual tools below. macOS Ventura / Sonoma / Sequoia / Tahoe · Apple Silicon and Intel

Tools

Each tool is self-contained with its own documentation. Click for details.

tor-proxy
Route macOS system traffic through the Tor network. Detects your active network interface, starts Tor, and configures the system SOCKS proxy. Supports enable, disable, status check, and identity renewal.
privacy network
brew install tor-proxy
chromium-browse
Headless browsing traffic generator for any Chromium-based browser. Drives Island, Chrome, Edge, Brave, Arc, Comet, Atlas, and others via Chrome DevTools Protocol. Randomized visits, link following, scroll simulation.
network automation
brew install chromium-browse
brew-upgrade
Automated Homebrew package updater. Fetches the latest index, shows outdated packages, upgrades formulae and casks, cleans up old versions, and logs everything with timestamps.
maintenance
brew install brew-upgrade
setup-claude
Automated Claude Code installation and configuration. Handles Homebrew cask install, directory structure, settings.json templating, status line style selection, and project management script deployment.
dev environment
brew install setup-claude

Installing

All tools live in the davidwhittington/mac-tools tap.

Examples
# Add the tap (one time)
brew tap davidwhittington/mac-tools

# Install individually
brew install davidwhittington/mac-tools/tor-proxy
brew install davidwhittington/mac-tools/chromium-browse
brew install davidwhittington/mac-tools/brew-upgrade
brew install davidwhittington/mac-tools/setup-claude
